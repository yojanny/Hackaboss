Elementos de bloque vs elementos de línea

Elementos de bloque

Los elementos de bloque generan saltos de línea. No tienen nada a los lados, el resto del contenido se coloca encima o debajo según el orden del HTML

Dentro de un bloque podemos meter practicamente cualquier otra etiqueta de bloque o de linea

Ejemplos: header, main, footer, section, article aside, div, p, h1/2/3/4/5/6...

Elementos de linea

Los elementos de linea ocupan lo minimo. No generan saltos de línea, pueden(y suelen) tener cosas a los lados

Dentro de un elemento de línea, solo puede haber elementos de linea

Ejemplos: span, strong, em, img, a...
